值绑定

v-model修饰指令

v-model用来在视图与model之间同步数据，但是有时候需要控制同步发生的时机，或者在数据同步到model之前将数据转换为number类型

  • lazy
    在默认情况下，v-model在input事件中同步输入框的值和数据，可以添加一个lazy特性，从而改到change事件中去同步。

    使用 .lazy 懒加载修饰符之后，只有在输入框失去焦点或者按回车键时才会更新 content 值。

  • number
    当传给后端字段必须是数值的时候，我们可以在v-model所在的控件上使用number指令，该指令会在用户输入被同步到model中时将其转换为数值，如果转换结果为NaN，则对应的model值还是用户输入的原始值

  • trim
    使用 ·.trim` 修饰符可以自动过滤掉输入框的首尾空格。
